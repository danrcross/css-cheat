CSS Snippet Cheatsheet

A cheatsheet for some useful CSS syntax

Flexbox Row

These properties will create a flexbox row layout:

           
                    .row {
                        display: flex;
                        flex-direction: row;
                        flex-wrap: wrap;
                    }

:hover Pseudo-Class Selector

This pseudo-class selector will change the font style to italic when the cursor hovers over the selected element(s):

                    .class:hover {
                        font-style: italic;
                    }

::after Pseudo-Element Selector

This pseudo-element selector will display "Wow!" in red text on a gray background after the selected element(s): 

                    .element::after {
                        content: "Wow!";
                        background-color: gray;
                        color: red;
                    }

Variables

The first snippet will assign the color "blue" to the variable "--maincolor". The second snippet will call that variable, and assign "blue" to all "p" elements.

                    :root {
                        --maincolor: blue;
                    }

                    p {
                        color: var(--maincolor)
                    }

Attribute Selectors

This style of selector ( using [ ] ) allows you to select elements that have been assigned certain attributes. This one will assign a bold font style to all anchor elements that contain an href= attribute ending in .css:

                a[href$=".css"] {
                    font-style: bold;
                }

"Nth Child" Selector (Even)

This pseudo-class will give all even-numbered list items a red font color:

                    li:nthchild(even){
                        color: red;
                    }
